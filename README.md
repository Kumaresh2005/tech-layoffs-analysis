# 📊 Tech Layoffs & Hiring Trends Automation

## 📌 Project Overview
This project automates data cleaning, preprocessing, reporting, and visualization for tech layoffs and hiring trends using Python and Pandas.

---

## 🚀 Features
- Data Cleaning Automation
- Missing Value Handling
- Duplicate Removal
- Trend Analysis
- Automated Reporting
- Data Visualization
- Downloadable Cleaned Dataset

---

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib

---

## 📂 Project Structure

tech-layoffs-analysis/
│

## ▶️ Run Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run:

```bash
python main.py
```
